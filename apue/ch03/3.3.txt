open() 함수를 호출하면 새로운 파일 테이블 항목을 생성한다. 문제에서 open() 함수 호출이 동일한 파일을 참조하기 때문에 두 파일 테이블 항목은 동일한 v-node 테이블 항목을 가리킨다. dup() 함수를 호출하면 기존의 파일 테이블 항목을 참조한다.
F_SETFD는 파일 기술자 플래그를 변경하는데 파일 기술자 플래그는 프로세스 테이블 항목으로 dup() 함수로 반환된 파일 기술자 fd2는 파일 기술자 fd1와 파일 테이블 항목을 공유하기 때문에 영향을 받는 파일 기술자는 fd1이다. 
이에 반해 F_SETFL은 파일 상태 플래그를 변경하며 파일 상태 플래그는 v-node 테이블 항목으로 파일 기술자는 fd1와 fd2가 공유한다. 따라서 영향을 받는 파일 기술자는 fd1와 fd2이다.
